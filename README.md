# Автомобільний каталог (React)

Простий вебдодаток для перегляду, фільтрації, додавання, редагування та видалення автомобілів.  

---

## Функціонал

- Перегляд списку автомобілів  
- Фільтрація за:
  - виробником
  - роком
  - кольором
  - об’ємом двигуна
  - діапазоном ціни
- Перегляд детальної інформації
- Додавання автомобіля
- Редагування автомобіля
- Видалення автомобіля
- Завантаження авто з публічного API (Tesla, Honda, Toyota, BMW)
- Різні параметри авто генеруються випадково
- Фото автомобілів через Unsplash
- Кнопка **«Очистити всі авто»**
- Автозбереження в LocalStorage
